display : block | inline | inline-block | none | contents | table | table-cell
    block level 요소 : 너비를 100% 가짐. 혼자서 한줄을 다 차지
    div, h1~h6, li, ul, ol, form, hr, fieldset, address, blockquote, figure, form, nav, main, header, article, aside

    inline level 요소 : 줄을 차지 하지 않고 자기 콘텐츠 영역만 차지, 한줄에 여러개의 인라인 레벨 요소 표시 가능
    span, a, img, input, label, button, strong, object, br, sub, sup, textarea, select

    inline-block 요소 : 한줄로 배치 가능, 요소를 inline으로 배치 하면서 내용에는 block 레벨 요소를 적용 (flot:left; 와 비슷)
